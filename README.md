# API + Excel Data Integration ETL Pipeline

## 📌 Project Overview
This project combines data from an Excel file and a live API to create a unified dataset.

## ⚙️ Features
- Extract data from Excel
- Fetch live data from API
- Merge datasets
- Perform data transformation
- Export final dataset

## 🛠️ Technologies Used
- Python
- Pandas
- Requests
- OpenPyXL

## ▶️ How to Run
1. Install dependencies:
   pip install pandas requests openpyxl

2. Run:
   python api_excel_etl.py

## 📊 Output
Final dataset:
final_combined_data.csv

## 💼 Use Case
Simulates real-world data integration tasks used in business analytics and reporting systems.
